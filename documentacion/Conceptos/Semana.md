# Semana

Periodo de referencia para las estimaciones. Inicia el **domingo anterior** y termina el **sábado** de dicha semana.

Es la ventana usada para filtrar el [[Consolidado diario]] y para acumular el [[Consolidado semanal]].

Relacionado: [[Consolidado diario]] · [[Consolidado semanal]] · [[Estimado diario]]
