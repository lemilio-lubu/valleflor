# La carga masiva debe crear las fincas faltantes

Hoy la [[Carga masiva]] falla con *"La finca X no existe en el sistema"* (p. ej. LIBERTAD, PIFO, PUEMBO). El comportamiento esperado es el contrario: si la [[Finca]] no existe, **la carga debe crearla**, porque justamente para eso sirve la carga.
