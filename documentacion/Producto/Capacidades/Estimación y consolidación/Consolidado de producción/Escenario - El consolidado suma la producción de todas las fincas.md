# Escenario — El consolidado suma la producción de todas las fincas

```gherkin
Dado dos fincas que producen el mismo ítem de catálogo en la misma semana
Cuando el administrador consulta el consolidado de esa semana
Entonces el ítem aparece en una sola fila con la suma de ambas fincas
```
