# Captura de producción

Capability operativa donde cada [[Responsable]] registra la producción real: abre las [[Semana]]s de trabajo y captura las [[Caja]]s de cada [[Color]] en la [[Plantilla diaria]], con cálculo automático de [[Tallo]]s.

Rol central: [[Responsable]].
