# Escenario — Las cajas se redondean a dos decimales

```gherkin
Dado un registro diario de un color
Cuando el responsable registra cajas con más de dos decimales
Entonces las cajas se redondean a dos decimales
Y los tallos se calculan sobre el valor redondeado
```
