# Gestión de semanas

**Descripción:** creación y mantenimiento de las [[Semana]]s de trabajo del [[Responsable]]. Al crear una semana se genera su [[Plantilla diaria]] para cada [[Color]] asignado.

> Como [[Responsable]]
> quiero crear y gestionar las semanas de trabajo
> para registrar la producción diaria de los colores que tengo asignados.

## Escenarios
- [[Escenario - Crear una semana genera la plantilla diaria]]
- [[Escenario - No se duplica una semana del responsable]]
- [[Escenario - Solo un responsable puede crear semanas]]
- [[Escenario - Un responsable elimina su propia semana]]
- [[Escenario - Las semanas se listan paginadas]]
